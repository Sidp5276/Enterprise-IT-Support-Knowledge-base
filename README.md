# Enterprise IT Support Knowledge Base

## Overview

This repository contains enterprise-style IT support troubleshooting tickets created for hands-on learning and interview preparation.

The goal of this repository is to simulate real Service Desk / IT Support workflows followed in enterprise environments.

Each ticket includes:

- User reported issue
- Environment details
- Initial analysis
- Troubleshooting steps
- Commands used
- Root Cause Analysis (RCA)
- Resolution
- Verification
- Preventive actions
- Technical concepts involved
